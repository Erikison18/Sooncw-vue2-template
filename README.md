## 开发和构建

建议使用 [Yarn](https://yarn.bootcss.com/) 作为依赖管理工具。

```bash
# 依赖安装
$ yarn
# 或
$ npm install

# 启动开发调试模式
$ yarn serve
# 或
$ npm run serve

# 使用指定环境（如：uat）的接口调试
$ yarn serve:prod
# 或
$ npm run serve:prod

# 构建用于生产的输出
$ yarn build
# 或
$ npm run build

# 基于 eslint 格式化工程代码
# 这将检查语法问题并修正
$ yarn lint
# 或
$ npm run lint
```
